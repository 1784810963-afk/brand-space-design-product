# 设计新闻Agent MVP - 配置和测试指南

## 功能概述

设计新闻Agent是AI设计助手的新功能，可以自动获取和分析科技公司的最新设计动态。

**支持的公司：** Apple、华为、蔚来、小米

**核心功能：**
- 用户输入公司名称，Agent自动识别并爬取新闻
- AI分析内容，生成结构化的设计动态报告
- 支持自然语言对话

## 配置步骤

### 1. 获取千问API Key

1. 访问 https://dashscope.aliyun.com/
2. 登录阿里云账号
3. 开通"模型服务灵积"（DashScope）
4. 在控制台获取API Key

### 2. 配置环境变量

在 `brand-space-web/.env.local` 文件中添加：

```bash
DASHSCOPE_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

**注意：** 如果文件不存在，请创建它。

### 3. 安装依赖（如果需要）

```bash
cd brand-space-web
npm install
```

## 使用方法

### 1. 启动开发服务器

```bash
cd brand-space-web
npm run dev
```

服务器将在 `http://localhost:3000` 启动

### 2. 访问AI助手页面

打开浏览器访问：`http://localhost:3000/ai-chat`

### 3. 使用设计新闻Agent

1. 点击"设计新闻Agent"模式卡片（📰图标）
2. 在输入框中输入公司名称，例如：
   - `apple`
   - `华为`
   - `华为最近有什么设计新闻？`
   - `蔚来汽车`
   - `小米手机`
3. 点击发送或按回车键
4. 等待10-30秒（爬取和分析需要时间）
5. 查看AI生成的设计动态报告

## 故障排查

### 问题1：千问API调用失败

**症状：** 报错"千问API调用失败"

**解决方案：**
1. 检查 `.env.local` 中的 `DASHSCOPE_API_KEY` 是否正确
2. 检查API Key是否有效（登录DashScope控制台验证）
3. 检查账户是否有足够的配额

### 问题2：爬取超时

**症状：** 长时间无响应或提示"爬取失败"

**解决方案：**
1. 检查网络连接
2. 目标网站可能暂时无法访问，稍后重试
3. 尝试其他公司（不同的URL可能可用性不同）

### 问题3：识别不到公司名

**症状：** 输入公司名后提示"目前支持的公司包括..."

**解决方案：**
1. 确保输入的是支持的公司名（Apple、华为、蔚来、小米）
2. 尝试不同的输入方式：
   - 中文名：华为
   - 英文名：apple
   - 组合：蔚来汽车、小米手机

### 问题4：返回内容不包含设计相关

**症状：** AI分析的内容不相关或没有设计新闻

**解决方案：**
1. 这可能是因为目标公司最近一周确实没有发布设计相关新闻
2. AI会在报告中说明这一情况
3. 尝试查询其他公司

## 性能指标

- **首次查询响应时间：** 10-30秒（包含爬取和AI分析）
- **爬取超时：** 30秒
- **支持的公司数：** 4个（Apple、华为、蔚来、小米）
- **爬取成功率：** 预期 >80%（至少一个URL成功）

## 技术栈

- **前端：** Next.js 16 + React 19 + TypeScript
- **AI模型：** 通义千问（qwen-plus）
- **爬取：** Next.js API Routes + fetch
- **状态管理：** React Hooks
