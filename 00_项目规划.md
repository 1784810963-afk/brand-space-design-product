# 品牌空间设计网站 - 实现计划

## 项目概述
为品牌空间设计团队构建一个 **Apple 风格** 的展示与互动平台，包含项目展示、设计标准展示和 AI 驱动的机器人咨询功能。

---

## 需求确认总结

### 核心功能
1. **空间项目展示页面** - 展示已落地的 10+ 个品牌空间项目
2. **设计标准展示系统** - 展示 9 大设计标准模块：
   - 1. 设计总则、2. 场地设置原则、3. 外观设计原则、4. 室内设计原则、5. 墙面/家具/软装
   - 6. 照明设计、7. 标识系统、8. 材料清单、9. 标准节点图集
3. **AI 机器人助手** - 三类机器人（并行实现）：
   - 设计趋势分析
   - 设计标准咨询
   - 案例获取

### 技术选型已确认
- **框架**: React + Next.js (SSR + 静态导出混合)
- **样式**: Tailwind CSS / 自定义 CSS (Apple 极简风格)
- **AI 方案**: 大模型 API（具体型号待定，可选：文心一言/Claude）
- **部署**: 待定（先完成开发）
- **用户范围**: 内外兼顾（内部员工 + 外部客户）

### UI 设计风格确认
- **配色**: 浅色主题（Light Theme）- 参考 Apple 官网
- **设计特点**:
  - 极简主义 - 大量留白和负空间
  - 清晰的信息层级 - 大标题 + 副标题 + CTA 按钮
  - 全屏分区块设计 - 每个内容模块占据整个视口
  - 高品质视觉 - 项目/产品图片为中心
  - 简洁导航 - 顶部导航栏
- **交互动画**:
  - ✅ Hover 效果（按钮、卡片）
  - ✅ Scroll-based 动画（页面滚动时触发）
  - ✅ Page transition（页面切换动画）
- **响应式**: 桌面端优先（后期考虑移动端适配）

### 已有资源
- ✅ 项目案例 - 10+ 个完整案例（图片、视频、描述）
- ✅ 设计标准文档 - 9 大模块已整理完成
- ❌ 设计趋势分析内容 - 需边做边补充

### MVP 策略
- **初期上线**: 2-3 个代表性案例 + 完整设计标准 + 基础机器人功能
- **后期迭代**: 补充更多案例 + 优化机器人智能度
- **目标上线**: 1-2 月内 MVP 上线
- **内容管理**: 暂不需要 CMS，直接修改代码配置文件更新内容

## 推荐实现方案

### 架构方案

#### 1. 项目结构
```
brand-space-design-web/
├── app/                           # Next.js App Router
│   ├── layout.tsx                # 全局布局 + 导航栏
│   ├── page.tsx                  # 首页 - 全屏分区块设计
│   ├── projects/                 # 项目相关路由
│   │   ├── page.tsx              # 项目列表页
│   │   └── [id]/page.tsx         # 项目详情页
│   ├── standards/                # 设计标准相关路由
│   │   ├── page.tsx              # 标准导航页
│   │   └── [category]/page.tsx   # 标准详情页
│   └── ai-chat/                  # 机器人相关
│       └── page.tsx              # AI 机器人页面
├── components/                   # 可复用组件
│   ├── Hero.tsx                  # 全屏英雄区块
│   ├── ProjectCard.tsx           # 项目卡片
│   ├── StandardSection.tsx       # 标准展示区块
│   ├── AIChat.tsx                # AI 对话组件
│   └── ...
├── data/                         # 数据配置（不需要 CMS）
│   ├── projects.ts               # 项目数据
│   ├── standards.ts              # 标准数据
│   └── constants.ts              # 常量配置
├── lib/                          # 工具函数
│   ├── api.ts                    # 大模型 API 调用
│   └── utils.ts
└── styles/                       # 全局样式
```

#### 2. 关键页面设计

**首页 (Home Page)**
- Section 1: Hero 区块 - 品牌标题 + CTA 按钮
- Section 2: 项目展示 - 精选 3 个代表性案例
- Section 3: 设计标准介绍 - 9 大模块概览
- Section 4: AI 机器人 - 三类机器人功能卡片
- Section 5: 底部导航 - 快速链接

**项目列表页 (Projects)**
- 网格/列表展示 10+ 项目
- 支持按标签/分类筛选
- 项目卡片：缩略图 + 标题 + 描述 + 查看详情

**项目详情页 (Project Detail)**
- 大图展示 + 项目信息
- 项目描述、特点、成果展示
- 相关案例推荐

**设计标准页 (Standards)**
- 导航页面展示 9 大模块
- 点击进入各模块详情（支持扩展内容）

**AI 机器人页 (AI Chat)**
- 聊天界面（三种机器人模式选择）
- 对话历史
- 快速提示词
